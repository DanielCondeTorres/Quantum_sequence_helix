Key Physics
U_C(γ) = e−iγZ rotates around Z.
Z-rotation never changes |α|² or |β|² — the probability stays flat. Only the relative phase φ = arg(α)−arg(β) shifts.
U_B(β) = e−iβX rotates around X.
X-rotation mixes |0⟩ and |1⟩ components — probability changes. The phase accumulated by U_C is now converted into amplitude.
